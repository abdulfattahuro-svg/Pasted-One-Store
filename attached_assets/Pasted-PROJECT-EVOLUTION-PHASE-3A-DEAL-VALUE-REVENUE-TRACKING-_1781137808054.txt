PART 1 — DEAL VALUE SYSTEM
==========================

Enhance leads table.

Add fields:

* dealValue
* currency
* expectedValue
* actualRevenue
* closedDealValue
* payoutAmount

Purpose:

expectedValue

Potential estimated deal value.

Example:

₦2,500,000 solar project

closedDealValue

Final deal value after closing.

Example:

₦2,200,000

actualRevenue

Revenue earned by business.

Example:

₦1,950,000

payoutAmount

Final affiliate payout amount.

===================================

Commission engine must support:

Fixed

Percentage

Hybrid

Examples:

Solar:

10% of deal value

Real Estate:

2% of sale

Insurance:

15% of premium

Hybrid:

₦5,000 + 5%

Logic:

If commission type = percentage:

Use:

closedDealValue

If unavailable:

fallback:

expectedValue

If commission type = hybrid:

calculate both:

fixed + percentage
